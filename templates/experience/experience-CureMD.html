AI Engineer

CureMD Pvt Ltd

Duration: Jul 2024 – Sept 2025

Credential: View Credential

Summary: Building AI features for healthcare, including predictive modeling, clinical documentation agents, multimodal document understanding, and real time voice bots shipped across internal workflows and piloted for broader deployment.

  • Modeled treatment outcomes for cholesterol patients using an XGBoost based regression pipeline to predict HDL, LDL, and Triglycerides from demographics, vitals, medical history, and treatment data.
  • Built an HPI Agent to automate the Subjective section of SOAP notes using a hybrid approach of complaint specific rule templates, RAG and Llama 3.1 8B.
  • Developed MediExtract, a multimodal AI pipeline to extract structured information from oncology lab documents such as scanned PDFs, tables, and handwritten reports. The system combined document classification, YOLO based table detection, PaddleOCR, and vision language models to accurately capture lab values, biomarker data, and key metadata for downstream clinical use.
  • Built an Interview Bot using livekit-meet, KOKORO TTS, Whisper large v3, Qwen 3 32B, and Silero VAD. Sessions were evaluated using post processed transcripts and video recording was handled through Egress.
  • Developed SmartDialer for RCM to automate provider verification calls with insurance IVRs including real time transcription, auto filling required data, and handing off to an RCM agent after verification.
  • Additional work included multimodal systems combining LLMs, computer vision, and speech along with API services built using FastAPI.
